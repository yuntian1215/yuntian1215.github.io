奇怪的梦

发表于 2023-08-12 更新于 2023-08-14

昨晚做了一个奇怪的剧情梦，屋主已经很久没有做过那么长的梦了，并且刚好梦结束后清醒，所以也是记得最清楚的一个梦了。后续和J导分析了一下，好像是自己一些负面情绪在潜意识跑出来了。

梦的主要场景是夏令营，主人公是我，高中数学老师，小学记忆中的“暴力女”，还有前c，J导也友请客串了一下我的梦境。为什么是夏令营呢， 梦里给我的解释是之前的笔试是算面试的一部分的，所以还要再笔试两次。呜呜，前一段时间这个笔试确实让屋主纠结了好几天。

然后考完第一场试后，去第二场考试选座位的时候就被一些很久未见的，但是被在小时候打上暴力标签的同学拒绝了，不过屋主已经十年没有和她们接触了…

这时候高中数学老师登场，告知我第一场发挥出色，把压轴写完分数溢出了，第二场已经不用考了。不过屋主的数学无论是高考还是竞赛都是差一点点，或许它们出现在梦里是一种执念所致？不过梦中都在肯定鼓励自己，很好嘿嘿！

最后是夏令营结束后要出去拍照，要走好一些路程。屋主就开始寻找前c的位置，然后就一直follow，看到好多次回眸，隐隐确定没认错人，但是最后合照靠近的时候，发现不是… 然后屋主就醒了，有点像背影是真的，人是假的。

虽然白天略微有点点恍惚，但晚上发现屋主最喜欢的偶像–S.H.E.昨晚合体了！唱了美丽新世界！Selina的嗓音有所起色，听到那句别再记着昨天，还有什么悲伤鼻子开始发酸，呜呜，希望小腰果能帮助公主妈妈重拾自信，让圆圆嗓音慢慢恢复吧！

家乡

发表于 2023-08-09

屋主七月底归家后，这一段时间简直幸福得快要昏古起了，每天都很开心！所以自然要多记录一下这样的开心时刻咯！

首先最开心的就是每天都可以午睡！由于学校宿舍隔音不好，所以屋主在学校一个月都不一定能睡成一次午觉。（虽然每天都要爬上床，但是很快就会被走廊上下楼的声音吵醒，由此对所有硬底拖鞋怨念加加）回家后，每天中午睡得都很安心，而且睡醒后还能享受老爹泡的蜂蜜和老妈买的石花冻解暑，真的好快乐！

其次是老家有好多好朋友，屋主今天和H哥一起学唱歌，明天去Y兄家追忆往事，后天和隔壁班一起看老师看电影唱k，最后实在闲着没事，再{慰问/骚扰}一下尚在远方实习和备考的SS。所有情绪都能很快分享出去，并且及时得到回应，算是一种难得的时刻吧~

然后是在家效率比在学校高😕。因为家楼下装修的原因，所以屋主白天就浪迹于图书馆和新华书店。图书馆可以自习，累了还可以摸书看，个人觉得是最佳的学习/工作地点。说到看书，屋主的阅读曲线从小学到大学是这样的/\/，不懂够不够抽象。简言之就是中学时代几乎没什么读书，或者说，读了，但都是小说，还不是经典小说（玄幻/言情/科幻…）到大学后渐渐意识到思想的缺失，思考的匮乏，于是乎慢慢开始向小学看齐…某天在新华书店看了半本书后，愈发体会到读纸质书的快乐，像是重新打开了某个开关。（像是在各种书中都能瞥到一部分自己，然后看到各种稀奇古怪的想法/回答后能进一步启发自己给出生活的解？）希望回学校后还能继续坚持！

最后是老家的天气蛮宜人的。尽管正值八月，酷暑炎炎，但是街边绿树成荫，遇见红绿灯时，只需往树下一靠，别提多凉快，屋主就经常骑着小电驴四处闯荡。下雨也非常会挑时候，通常就是饭前示意性下一下，你真要吃饭的时候它就停了，起一个降温上的作用hhh 当然也可能是屋主在家心情太好了，看啥啥美~

不过很快就要返校了，屋主表示：o(一︿一+)o 出于好玩报了个训练营，如果过了的话八月还剩的节目就是包邮区三角游了，希望一切顺利吧！

ps：“我要我的身体和心一样干净 尽管这样，并不是为了见到你 ”

七月

发表于 2023-07-30 更新于 2023-08-04

三年之期已到，屋主又在这个毕业季前前夕开始伤感起来…

转专业和疫情的阵痛让屋主在短暂的大学时光里没有认识到很多人，而屋主比较喜欢… 喜欢认识陌生人？又或者是不喜欢身边环境骤变，所以对于各奔东西这种事免不了带些离愁别绪，不过比三年前好多了~（伤感缘由还包括假期爱玩没什么干活导致的自我认同缺失hh…)

不过emo只持续了一小会，在努力思考＋听了John导反复的“You should do what you really enjoy”后，还是暂时点破了一些自设的情绪陷阱，对于未来的航向多少有了一点点基本感觉和想法。期间感谢J‘导’和D兄的分析和疏导，真的相当lucky，在这个阶段还能遇见一些愿意理会屋主情绪的好人~

然后就是摸鱼着做了一些暑期实习的工作，不懂王导未来会不会看到，希望留屋主一条命吧hh…

最后是一些很小品的事情… 比如认识到自己在独立过程上道阻且长，目前心理还是很重度依赖别人这件事… 比如因此认识到牡丹爆冲酿成大祸不知是否解释清楚希望菠萝多年后不要暴打屋主这件事… 还有比如在J‘导’两个直立柱子靠近理论指导下还是忍不住crush陌生人，然后发现别人用的是情头这件事… 总觉得这一个学期在参演某个喜剧，可是内核却有点荒诞悲凉…

嘿，这些就能让屋主破防吗？不可能！看看远方的宣讲大使和声乐课吧！应该会让屋主的八月非常明媚的哈哈哈哈哈哈！(★ ω ★)

ps：D兄当时问了屋主觉得坍缩这一说法自然吗，然后说现在更被接受的观点应该是叠加态是由各种态占比组成的，而观测后“坍缩”现象解是因为观测实验本身会造成无法理解的现象。而由于基于坍缩的说法没有办法演化出有用的结论，所以被主流放下了。屋主觉得好有意思，觉得值得留在这个七月记里~

后续：七月的最后一天和H好友去学了一节声乐课，好喜欢，感觉了解乐理知识的过程有种莫名的快乐和自我满足，能够打破屋主奇怪的情绪状态嘿嘿。然后八月第一天屋主的好友返校前邀了一次ktv，呜呜，又捕获到一首好听的歌——《恶作剧》！屋主真的很喜欢各种台偶的配曲，真的很甜，很有一种命中注定的遥远的奇迹般的美感。不过屋主之后又突然一抹（emo）了一下，因为觉得还是有一些陌生人可能看上去挺有趣，但屋主似乎已经没有什么机会/缘分去认识了，似乎有点可惜。（就是朋友的角度嗷，屋主现在心若冰清！）不过综合以往所有的感觉总结，会有点像前面那首歌里面的词“没有人像我和陌生人的爱恋”😂

不过在和H,Y两位好友聊过后，他们都有过相似的感觉，好了许多。然后又上了一节声乐课，芜湖，屋主的音准有在渐渐变好，感觉似乎某个小心愿有了一定实现的可能性！嘿嘿！

Why Blog

发表于 2023-07-12

博客，上古时期也被唤为部落格，简单搜索了一下主要可以用来写网络日记和技术文档。

屋主作为一个内心剧场丰富的铁血硬汉喵，对于分享心情有着很大的情感需求。之前曾用了一段时间的公众号，但由于透露了过多的私人信息，只能在几位损友之间传播。所以屋主自然是要另辟蹊径，找一些能写的地方续谱华章，这就瞄上了博客。

损友之所以为损友，就是他们可以完全忽视你的post…不过诚如J导所言，找到能回应你分享日常的朋友是一种幸运，所以不要对朋友这一身份有那么高的期待和依赖。

其实在大二搭完某个破烂博客网页后就一直很想建一个人博客写写日记，小范围的分享一些心情。无奈，太懒＋想得多做得少这一组合拳让屋主拖到了现在才开始行动。悟以往之不谏，知来者之可追，克服内心恐惧怠惰，多做行动派也是屋主成长中关键的一环。所以往后就多见多想多写，等它累积到一定量后，应该会有一定质变的吧🌼
